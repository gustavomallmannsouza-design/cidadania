# cidadania# Cidadania Digital e Inteligência Artificial
**Subtítulo:** O Impacto das Deepfakes e da Desinformação na Sociedade

## 🎯 Objetivo do Projeto
Este site foi desenvolvido como atividade de Recuperação Prática para o componente curricular de Educação Digital e IA. O principal objetivo da página é conscientizar os estudantes e a comunidade escolar sobre os perigos da manipulação midiática automatizada por ferramentas de Inteligência Artificial, ensinando métodos práticos de identificação de fake news e diretrizes éticas para navegação segura na internet.

## 🛠️ Tecnologias Utilizadas
- HTML5 (Estruturação semântica)
- CSS3 (Estilização com Layout Flexbox e Responsividade com Media Queries)
- JavaScript Vanilla (Lógica do Quiz iterativo e funcionalidade de Modo Escuro)

## 🤖 Uso de Inteligência Artificial (Declaração de Ética)
Para a geração das estruturas lógicas básicas de programação e revisão do conteúdo textual, foi utilizada a Inteligência Artificial.
- **Ferramenta:** Copilot / ChatGPT
- **Prompt Utilizado:** *"Gere um esqueleto estrutural limpo de site institucional em HTML5 semântico, folha de estilo em CSS responsiva com Flexbox para modo escuro nativo e um arquivo de script JS nativo contendo um validador de quiz escolar focado no tema Cidadania Digital e combate a Deepfakes."*
